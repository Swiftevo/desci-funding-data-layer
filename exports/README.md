# Exports Layer

This folder contains public-facing and machine-readable exports from the DeSci Funding Data Layer.

---

# Structure

## public/

Latest stable exports intended for:
- APIs
- AI ingestion
- ecosystem tooling
- public datasets
- lightweight integrations

These files may be updated over time.

---

## releases/

Versioned export releases.

Used for:
- stable archival exports
- ecosystem references
- historical distribution
- reproducible research

---

# Notes

Snapshots are stored separately under:

/snapshots/

The exports layer focuses on accessibility and interoperability rather than historical preservation.
